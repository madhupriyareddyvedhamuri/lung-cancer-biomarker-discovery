# Lung Cancer Biomarker Discovery Using Gene Expression Analysis

## Project Overview
This bioinformatics project analyzes real lung cancer gene expression data to identify potential biomarkers associated with lung cancer.

## Objectives
- Analyze gene expression datasets
- Identify differentially expressed genes
- Visualize biological patterns
- Discover potential biomarkers

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Project Structure
- data/
- notebooks/
- results/
- figures/
- src/

## Analysis Performed

- GEO lung cancer dataset acquisition
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Gene expression distribution analysis
- Top highly expressed gene identification
- Top variable gene identification
- Result export to CSV
